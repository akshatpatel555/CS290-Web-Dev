    Locations

    There are 3 different types of main locations for CSS styles. These are inline, internal and external. Inline style are dirctly embeded within HTML element. Internal style are included in the "head" section of the HTML element. And External style is used using an external ".css" file. For example, for Assignment 3 we use an External style in the form of "main.css".

    Selectors

    CSS Selectors are patters that are used to select different elements that a user wants to style. Different types of selectors are class, id, attribute, psuedo. Class selectors are used to all elements with the class "intro". Id selectors are used to select elements with a particular name.

    Box Model

    In CSS a box model is used as a fundamental layout design used to create a box around cetain elements with "content", "padding", "border" and "margins".

    Inheritance & Specificity

    Inheritance in CSS means that a certain styling properties of a parent element are also inherited by child elements.
