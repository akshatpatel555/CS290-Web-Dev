    Definition of Terms

    Frontend design term definitions:

    Website: Website is a page on the internet that hosts a collection of information about a certain topic or ideas.
    Web Application: A web application is a software that is hosted on a website.
    Web Browser: A web browser is a software that is designed to allow users to visit and access the internet and websites.
    Web Server: A server is a software that allows to host a website and other applications on the internet.
    HTTP Request: A http request is the way that allows users to access a website through a private server.
    HTTP Response: A http response is made by the server in order for a user to get access to a website.

    Efficient:
    Users can accomplish their goals quickly and easily.

    Effective:
    Users can complete their tasks accurately.

    Engaging/Enjoyable:
    Users are left appealed and come back often.

    Error-free:
    Users are met with low chances of roadblocks.

    Easy to navigate:
    Users understand how to locate their goal.

    The purpose of a page layout is to provide a proper sturucture and organization to a webpage.


    Tag Description
    html Defines the root of an HTML document.
    head Contains metadata and links to external resources.
    body Contains the content of the document visible to users.
    header Represents introductory content, typically at the top of a page.
    nav Contains navigation links for users to navigate within the website.
    footer Contains footer content such as copyright information or contact details.
